## 1. 目的

这份手册是 Unity2Foxglove 的用户入口，用于帮助你安装 Unity2Foxglove、连接 Unity 与 Foxglove、录制或回放 MCAP 文件，以及构建 IL2CPP Player。

## 2. 阅读路径

大多数用户可以按这个顺序阅读：先安装 package，再连接 Foxglove，然后发布 Unity 数据，使用 Parameters 和 Services，录制/回放 MCAP，构建 Player，最后按症状排查常见问题。

> [!NOTE]
> SDK 核心面向 Unity 6000.0 LTSC 或更新版本（兼容 6000.0.74f1 LTSC）。不支持 Unity 2022。仓库自带的 `Unity2Foxglove` demo project 基于 Unity 6000.3.14f1 LTSC 开发和测试。

## 3. 快速开始

- [01_前置条件](01_前置条件.md)：安装 Unity、Foxglove Desktop 和可选开发工具。
- [02_安装与快速上手](02_安装与快速上手.md)：安装 package，并看到 `/tf`、`/scene` 和 `/unity/camera`。
- [样例和演示项目（英文）](../en/03_Samples_and_Demo_Project.md)：在打开 Unity 前选择合适的项目或 package sample。
- [04_Foxglove操作指南](04_Foxglove操作指南.md)：使用 Foxglove Desktop 的面板和布局。
- [05_基础可视化](05_基础可视化.md)：导入最小 package sample。

## 4. 运行时控制

- [06_Parameters与Services](06_Parameters与Services.md)：编辑 `/cube/color`、`/cube/scale`，并调用 `/cube/reset_pose`。
- [07_FoxRun自动发布](07_FoxRun自动发布.md)：使用 `[FoxRun]` attribute 发布调试 topic。
- [Inspector Reference（英文）](../en/12_Inspector_Reference.md)：理解主要 Unity component 的 Inspector 字段。

## 5. 录制、回放和构建

- [08_MCAP录制回放](08_MCAP录制回放.md)：录制 Unity 数据，在 Foxglove 中打开 MCAP，并在 Unity 中回放。
- [09_IL2CPP构建](09_IL2CPP构建.md)：使用 `Scripts/build_tools/unity_il2cpp.py` 构建并验证 standalone Player。

## 6. 进阶阅读

- [10_架构说明](10_架构说明.md)：了解 runtime、protocol、MCAP 和 FoxRun 内部结构。
- [11_常见问题排查](11_常见问题排查.md)：按症状排查问题。
- [Schema Coverage（英文）](../en/13_Schema_Coverage.md)：了解官方 protobuf schema 覆盖情况和 typed publisher 边界。

## 7. 项目选择

- 使用你自己的 Unity 项目：适合把 SDK 接入已有应用。
- 使用 `Samples~/BasicVisualization`：适合最小 package sample 验证。
- 使用 `Samples~/FullDemoVisualization`：适合完整 package 功能体验，包括 Parameters、Services、FoxRun、MCAP、Input System 和 URP。
- 使用 `Unity2Foxglove`：适合仓库开发、人工验收和构建测试。
