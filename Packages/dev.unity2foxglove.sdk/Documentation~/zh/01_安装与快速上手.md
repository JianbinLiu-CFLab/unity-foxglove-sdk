# 1. 快速开始

## 谁应该阅读本文

如果你是第一次将 Unity2Foxglove 添加到 Unity 项目，请阅读本文。

## 你将完成以下操作

你将安装该包，添加 `FoxgloveManager`，发布一个 Transform，可选发布一个场景方块和相机图像，并连接 Foxglove Desktop。

> [!TIP]
> 如果你尚未安装 Unity 或 Foxglove Desktop，请先从 [00 准备工作](00_前置条件.md) 开始。

## 1.1 安装包

### 1.1.1 从本地包安装

1. 打开 Unity。
2. 打开 **Window > Package Manager**。

![open-package-manager](../Pictures/01/open-package-manager.png)
<figcaption>图1: 打开 Package Manager</figcaption>

3. 点击 **+ > Add package from disk...**，或选择下方的"Install package from git URL"并输入以下 URL：

```http
https://github.com/JianbinLiu-CFLab/unity-foxglove-sdk.git?path=/Packages/dev.unity2foxglove.sdk
```

![package-manager-add-menu](../Pictures/01/package-manager-add-menu.png)
<figcaption>图2: Package Manager 添加菜单</figcaption>

4. 选择 `Packages/dev.unity2foxglove.sdk/package.json`。

![select-package-json](../Pictures/01/select-package-json.png)
<figcaption>图3: 选择 package.json 文件</figcaption>

5. 等待 Unity 解析依赖。

![package-imported-confirmation](../Pictures/01/package-imported-confirmation.png)
<figcaption>图4: 包导入确认</figcaption>

### 1.1.2 备选方案：编辑 `manifest.json`

此方法不推荐常规使用——Unity 可能会将相对路径 `file:` 重写为绝对路径，导致结果不可移植。除非你有特殊原因需要编辑 manifest，否则建议使用上述 Package Manager UI。

```json
{
  "dependencies": {
    "dev.unity2foxglove.sdk": "file:../../Packages/dev.unity2foxglove.sdk"
  }
}
```

根据项目与包之间的相对位置调整路径。

## 1.2 添加服务器组件

1. 创建一个空 GameObject，命名为 `Foxglove`。
2. 添加 **FoxgloveManager** 组件。

![add-foxglove-manager](../Pictures/01/add-foxglove-manager.png)
<figcaption>图5: 添加 Foxglove Manager 组件</figcaption>

3. 首次测试时保持默认设置：
   - Host: `127.0.0.1`
   - Port: `8765`
   - Start On Enable: 启用
   - Coordinate Mode: `LeftHand`

![foxglove-manager-default-settings](../Pictures/01/foxglove-manager-default-settings.png)
<figcaption>图6: Foxglove Manager 默认设置</figcaption>

点击 **Play**，Unity Console 应显示服务器已启动的消息，地址为 `ws://127.0.0.1:8765`。

![websocket-server-started](../Pictures/01/websocket-server-started.png)
<figcaption>图7: WebSocket 服务器启动确认</figcaption>

## 1.3 发布 Transform

1. 创建一个 Cube。
2. 为 Cube 添加 **Foxglove Transform Publisher** 组件。

![add-transform-publisher](../Pictures/01/add-transform-publisher.png)
<figcaption>图8: 添加 Transform Publisher 组件</figcaption>

3. 首次测试时使用以下值：
   - Topic: 留空或使用 `/tf`
   - Parent Frame Id: `unity_world`
   - Child Frame Id: 留空则使用 GameObject 名称
   - Publish Rate Hz: `10`

![transform-publisher-tf-config](../Pictures/01/transform-publisher-tf-config.png)
<figcaption>图9: Transform Publisher `/tf` 配置</figcaption>

在 Unity 处于 Play Mode 时移动或旋转 Cube。

## 1.4 连接 Foxglove Desktop

1. 打开 Foxglove Desktop。
2. 点击 **Open connection**。
3. 选择 **Foxglove WebSocket**。
4. 输入 `ws://127.0.0.1:8765`。
5. 点击 **Open**。

确认 `/tf` 出现在 Topics 面板中。要查看原始 Transform 消息，请添加一个 **Raw Messages** 面板并选择 `/tf`。要可视化坐标帧，请添加一个 **3D** 面板。

![foxglove-tf-topic-connected](../Pictures/01/foxglove-tf-topic-connected.png)
<figcaption>图10: Foxglove 中 `/tf` Topic 连接</figcaption>

预期的 topics：

- `/tf`，schema 为 `foxglove.FrameTransform`

![tf-frametransform-topic](../Pictures/01/tf-frametransform-topic.png)
<figcaption>图11: `/tf` FrameTransform Topic 验证</figcaption>


在 Unity 处于 Play Mode 时，使用 Move Tool 或 Rotate Tool 在 Scene 视图中移动或旋转 Cube。Transform 的变化会立即发布到 `/tf`。

![cube-transform-update-foxglove](../Pictures/01/cube-transform-update-foxglove.png)
<figcaption>图12: Cube Transform 实时更新</figcaption>

## 1.5 可选：发布场景方块

如果你希望 Foxglove 的 3D 面板显示简单的方块图元而不仅仅是帧变换，请为 Cube 添加 **Foxglove Scene Cube Publisher** 组件。

![add-scene-cube-publisher](../Pictures/01/add-scene-cube-publisher.png)
<figcaption>图13: 添加 Scene Cube Publisher 组件</figcaption>

推荐的首次测试值：

- Topic: `/scene`
- Frame Id: 留空则复用对象/帧名称
- Color: 绿色
- Size: `(1, 1, 1)`

![scene-cube-publisher-config](../Pictures/01/scene-cube-publisher-config.png)
<figcaption>图14: Scene Cube Publisher `/scene` 配置</figcaption>

回到 Foxglove，添加一个新面板。

![foxglove-add-panel](../Pictures/01/foxglove-add-panel.png)
<figcaption>图15: 添加 Foxglove 面板</figcaption>

选择 3D 面板。

![select-3d-panel](../Pictures/01/select-3d-panel.png)
<figcaption>图16: 选择 3D 面板</figcaption>

预期的 topics：

- `/scene`，schema 为 `foxglove.SceneUpdate`

如果你在 3D 面板中看不到方块，请找到左侧面板 Topics 列表中的 `/scene`，点击可见性图标以启用它。

![enable-scene-topic-3d](../Pictures/01/enable-scene-topic-3d.png)
<figcaption>图17: 在 3D 面板中启用 `/scene` Topic</figcaption>
## 1.6 可选：发布相机图像

1. 选择一个 Unity Camera。
2. 添加 **Foxglove Camera Publisher** 组件。

![add-camera-publisher](../Pictures/01/add-camera-publisher.png)
<figcaption>图18: 添加 Camera Publisher 组件</figcaption>

3. 使用以下设置：
   - Topic: `/unity/camera`
   - Frame Id: `unity_camera`
   - Publish Rate Hz: `10`
   - Width: `640`
   - Height: `480`
   - JPEG Quality: `70`

![camera-publisher-config](../Pictures/01/camera-publisher-config.png)
<figcaption>图19: Camera Publisher `/unity/camera` 配置</figcaption>

回到 Foxglove，添加一个新面板。

![foxglove-add-camera-panel](../Pictures/01/foxglove-add-camera-panel.png)
<figcaption>图20: 添加相机可视化面板</figcaption>

选择 Image 面板。

![select-image-panel](../Pictures/01/select-image-panel.png)
<figcaption>图21: 选择 Image 面板</figcaption>

预期的 topics：

- `/tf`，schema 为 `foxglove.FrameTransform`
- `/scene`，如果你添加了 Scene Publisher，schema 为 `foxglove.SceneUpdate`
- `/unity/camera`，如果你添加了 Camera Publisher，schema 为 `foxglove.CompressedImage`

在 Unity 处于 Play Mode 时，使用 Move Tool 或 Rotate Tool 在 Scene 视图中移动或旋转 Cube。观察方块在 Foxglove 的 3D 面板中移动，并在各面板中查看位置更新。

![foxglove-live-updates](../Pictures/01/foxglove-live-updates.png)
<figcaption>图22: Foxglove 实时更新验证</figcaption>

## 1.7 成功的标志

- Foxglove 的 **Topics** 面板列出了你的 Unity topics。
- **3D** 面板可以显示 Cube 的帧或图元。
- **Image** 面板可以显示 `/unity/camera`。
- 在 Unity 中移动 Cube 会实时更新 Foxglove。
